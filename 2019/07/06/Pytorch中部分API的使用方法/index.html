Pytorch中基本API的使用方法

发表于 2019-07-06 分类于 pytorch Disqus：

最近在学习Pytorch，学习的过程中将使用到的API总结一下。

torch.zeros

返回一个全为标量 0 的张量.


类似方法还有
torch.randn(2,3)随机一个两行三列的tensor
torch.argmax() 获取最大位置
torch.matmul()两个矩阵相乘，支持numpy的broadcast机制

nn.embedding

随机初始化生成一个VN得矩阵，每一个1N维都代表一个词向量
这里面有两个重要参数：词汇表的大小V和词嵌入的维度N

nn.linear

对输入的数据进行线性变换

nn.utils.rnn.pack_padded_sequence&nn.utils.rnn.pad_packed_sequence

nn.utils.rnn.packpadded_sequence(input, lengths, batch_first=False, enforce_sorted=True)
将一个填充过的变长序列压紧
nn.utils.rnn.pad_packed_sequence(sequence, batch_first=False, padding_value=0.0, total_length=None):
将encoderoutputs在转换为Variable类型，得到的代表各个句子的长度，相当于解压
这里有一篇很好的文章说明了这两个方法的含义 https://zhuanlan.zhihu.com/p/34418001

model.train()

切换到训练模式，主要是针对model 在训练时和评价时不同的 Batch Normalization 和 Dropout 方法模式
train(mode=True)
parameters(memo=None) 返回一个 包含模型所有参数 的迭代器，一般用作optimizer的参数

model.eval()

切换到预测模式，主要是针对model 在训练时和评价时不同的 Batch Normalization 和 Dropout 方法模式

forward()

如果类继承自nn.Module，则必须实现该方法

torch.optim

优化器包，在初始化一个优化器的时候需要传入需要训练的参数，并且所有的optimizer都实现了一个step()方法来更新参数
